backend

Backend keel on see, mida kasutab server sinu kasutatavas programmis või veebilehel andmete protsessimiseks ja andmebaasidega suhtlemiseks, hoiab keskkonda rünnakute eest kaitstuna või suhtleb teiste seadmetega jms. Backend on vundament mille peale saab kõike ehitada ja tänapäeval on eriti oluline, et sellele ei jäetaks mingeid auke sisse, mida hiljem parandada on väga keeruline.

RUBY

Pealtnäha lihtne keel, aga piisavalt kompleksne seestpoolt nagu on ka inimene. Ruby keel on funktsionaalne ja samas kergesti loetav. Arendati algselt backend keeleks, aga on aastate jooksul arenenud sedaviisi, et nüüd on võimalik arendajal kirjutada sellega valmis nii frontend kui ka backend. Seda võimalust kasutavad nüüd mitmed suured firmad nagu ka Github. Ruby keeles on kogu sisestatud informatsioon objekt s.t kõike on võimalik muuta enda nägemuse järgi. Näiteks saad korrutamist ka ümber defineerida sõnaga * -> pluss.

PHP

Keel on loodud veebiarenduseks ja on üks esimesi neist. Kliendi päringud töödeltakse veebriserveris ja saadetakse tagasi HTML failid kliendile. Selle syntax on sarnane Java-ga, mis peaks Tartu Ülikooli tudengile lõpetamise ajaks üsnagi selge olema. Kuna tegemist on vana keelega, siis seda kasutavad ka seni suured hiiud nagu Wikipedia, Wordpress ja ka erinevad frameworkid. PHP tugevus on tema ühilduvus erinevate andmebaasidega: MySQL, PostGreSQL. PHP logoks on elevant, mis tuleneb prantsuse keelsest sõnast éléphant -> elePHPant ja seepärast elevant on ka PHP maskott.

JAVA

Java on tuntud kui objekt-orienteeritud programmeerimise keel, mis on laialt kasutusel tarkvara arenduses. Javat kasutatakse ka väga palju programmeerimse õpetamisel. Keelel on suur kogu erinevaid raamistike ja teekide kogu. Javat on võimalik jooksutada üldiselt kõikjal ja väga erinevate asjade jaoks ka mängude arendamiseks või kiipkaarti lahendustes. Javat kasutatakse ka suure jõudlusega andmetöötluses andmete salvestamiseks ja protsessimiseks.
